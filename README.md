# 술도둑 🍺

### 목표 
오늘의 기분과 먹을려는 술의 종류를 입력하면 OpenAI를 이용하여 채팅으로 술안주를 추천해주는 기능 개발<br>

### 기술스택

- HTML
- CSS
- Node
- Express
- OpenAI API
